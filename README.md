# 🏛️ Civ Strategy Game

A 4X Civilization Strategy Game built for web, playable on both **Mobile** and **Desktop** devices.

## 🎮 Features

- **4X Gameplay**: Explore, Expand, Exploit, Exterminate
- **Version Control System**: Compare game iterations easily
- **Dynamic World Map**: 10x10 procedurally generated terrain
- **Resource Management**: Cities, Power, Territory, Story Progress
- **Power-Up System**: Recruit Warriors, Archers, Fortresses, Cannons, Libraries
- **Boss Civilizations**: Face challenging enemy civilizations
- **Story Progression**: 10-chapter narrative campaign
- **Save/Load System**: Persistent game state
- **Responsive Design**: Works on mobile and desktop browsers

## 🚀 Quick Start

### Access the Game

Open the game in your browser:

```
https://calzado1997.github.io/civ-strategy-game/
```

Or upload the `civ_strategy_game` folder to any web server.

## 📱 Controls

| Button | Action |
|--------|--------|
| ➡️ Next Turn | Advance game turn, gain resources |
| 🎁 Power-Up | Recruit a random power-up (+5 Power) |
| 📖 Story Chapter | Advance story progression |
| 👹 Check for Boss | Trigger boss encounter |
| 💾 Save | Save current game state |

## 🔄 Version Control

The game includes a built-in version selector that allows you to:
- Switch between different game versions
- Compare iterations and see how the game evolved
- Test new features without losing progress
